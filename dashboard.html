Differential Equations

Course Description A differential equation is an equation which contains one or more terms and the derivatives of one variable (i.e., dependent variable) with respect to the other variable (i.e., independent variable) dy/dx = f(x) Here “x” is an independent variable and “y” is a dependent variable. For example, dy/dx = 5x.
